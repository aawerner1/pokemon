# Pokemon

### Dependências
 - ```Angular 8```
 - ```npm```

### Orientações

Primeiramente, faça o clone do projeto no seu computador e acesse a pasta *pokemonProject*.

Depois instale as dependências utilizando o comando:<br>
``` shell 
npm i 
```
Após isso, rode a aplicação usando o comando:<br>
``` shell  
ng s
```
